# 👩‍💻슬기로운 컴소생활

<br>

- **프로젝트 개요** : 학생들의 인트라넷 정보를 통해 재학생들의 편리한 학교생활을 목적으로 구현한 프로젝트
- **주요 기능**
  - 자유 게시판
  - 중고 거래 게시판
  - 졸업 요건 확인
  - 마이페이지
- **주요 기술**
  - BibleBot-scraper
  - MultipartFile
  - scheduling
  - Rest API
  - Intranet Sesstion
